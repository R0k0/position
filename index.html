All what you need know for now is that "body" element is "display: static" by default.
Every element in which you want to have another element. Must be "display: relative".
If you want second element to be relative on parent ("display: relative").
That second element must have some container for example "div" element which will be "display: absolute",
and in that container you actualy put second element with margins and other stuff.
Element that have "display" property, margin will not work.
So you need to add some position "left:" or/and "right" or/and "top:" or/and "bottom" to pozicionate
container element and if you wish add some "padding" to container, but the best will be to
set "height: auto;" and "width:auto;" to auto. Look to my code you will see.
